# Candidate Hiring Check (C++)

A simple console program in C++ that checks
whether a candidate meets the basic requirements for employment.

## Description
The program asks the user for:
- age
- work experience (in years)
- possession of a category D driver's license

Based on the data entered, the program displays the result:
whether the candidate is hired or not.

## Hiring conditions
The candidate is hired if:
- age 22 or older
- work experience 2 years or more
- has a category D driver's license
